# Simple JavaScript Function Testing

This example demonstrates fundamental software testing principles within a web context. It features a simple JavaScript function (`calculateDiscount`) that simulates a common web application logic. The code includes a basic, self-contained testing framework to verify the function's correct behavior with various inputs and its error handling for invalid data, illustrating how to ensure code quality and reliability.

## Language

`html`

## How to Run

1. Save the code as `index.html`.
2. Open `index.html` in your web browser.
