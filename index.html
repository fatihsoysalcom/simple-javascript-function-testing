Basit Web Uygulaması Test Örneği

Bu sayfa, bir web uygulamasının temel JavaScript fonksiyonunun nasıl test edilebileceğini göstermektedir. Yazılım testinin temel prensiplerini, yani bir fonksiyonun beklenen şekilde çalışıp çalışmadığını ve hata durumlarını doğru yönetip yönetmediğini doğrulamayı amaçlar.

Uygulama Mantığı: İndirim Hesaplayıcı

Aşağıdaki JavaScript fonksiyonu, verilen fiyat ve indirim yüzdesine göre yeni fiyatı hesaplar. Bu, bir e-ticaret uygulamasında sıkça karşılaşılan bir işlevdir.


function calculateDiscount(price, discountPercentage) {
    if (typeof price !== 'number' || typeof discountPercentage !== 'number') {
        throw new Error("Fiyat ve indirim yüzdesi sayı olmalıdır.");
    }
    if (price < 0) {
        throw new Error("Fiyat negatif olamaz.");
    }
    if (discountPercentage < 0 || discountPercentage > 100) {
        throw new Error("İndirim yüzdesi 0 ile 100 arasında olmalıdır.");
    }

    const discountAmount = price * (discountPercentage / 100);
    return parseFloat((price - discountAmount).toFixed(2)); // Küsürat hatalarını önlemek için
}
